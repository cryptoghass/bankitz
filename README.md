• PoW Algorithm: Lyra2Z

• PoW Blocks: 1 - 130000  
• PoS Blocks: Starting from 130001

• PoW Ending: ~ ca. 180 Days (Estimated: Feb 2019)  
• Masternode Requirements: 10,000 BKZ or 20,000 BKZ  
• Maturity: 41 Confirmations
